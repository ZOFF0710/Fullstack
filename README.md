本プロジェクトは、Node.js・Express・PostgreSQL・Drizzle ORMを用いて構築されたREST APIと、React・TypeScriptを使用したモダンなフロントエンドを統合したフルスタック商品管理アプリです。
TanStack Queryによるデータ取得・キャッシュ管理や、Tailwind CSS・DaisyUIによる洗練されたUIを採用し、実践的な開発構成を実現しています。
バックエンドからフロントエンドまでの完全な連携を通じて、実際のWebアプリケーション開発のベストプラクティスを学ぶことができます。

📂 무료 오픈소스 코드

🧪 .env 설정
Backend (/backend)
PORT=3000
DATABASE_URL=<YOUR_DB_URL>
NODE_ENV=development

CLERK_PUBLISHABLE_KEY=<YOUR_CLERK_PUBLISHABLE_KEY>
CLERK_SECRET_KEY=<YOUR_CLERK_SECRET_KEY>

FRONTEND_URL=http://localhost:5173

Frontend (/frontend)
VITE_CLERK_PUBLISHABLE_KEY=<YOUR_CLERK_PUBLISHABLE_KEY>

VITE_API_URL=http://localhost:3000/api

🔧 Backend 실행 방법
cd backend
npm install
npm run dev

🔧 Frontend 실행 방법
cd frontend
npm install
npm run dev
